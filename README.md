# RedesNeuronales

En este proyecto entrené y evalué modelos de redes neuronales para clasificar dígitos del 0 al 9 usando MNIST y un conjunto propio de 50 imágenes (5 por dígito).

Además, implementé la toma de predicciones en tiempo real con cámara.

Información del conjunto de datos

* MNIST: imágenes en escala de grises 28×28, un canal
* Valores 0–255 (normalizados a [0,1])
* 60,000 ejemplos de entrenamiento y 10,000 de prueba
* Etiquetas enteras 0–9.
* Imágenes propias (50): fotos de dígitos con iluminación y encuadres reales, 5 para cada dígito.
